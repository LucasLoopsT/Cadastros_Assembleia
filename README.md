## 📌 Visão Geral

Este projeto é uma **aplicação web fullstack** desenvolvida para gerenciar membros de uma igreja de forma eficiente.

O sistema conta com uma **API RESTful** e um **painel administrativo**, permitindo que administradores realizem autenticação segura, gerenciem dados e visualizem informações através de relatórios.

Além disso, este projeto marcou meu **primeiro contato com TypeScript**, onde apliquei conceitos importantes de arquitetura e boas práticas de desenvolvimento.

---

## ✨ Funcionalidades

### 👥 Gerenciamento de Membros
- CRUD completo de membros  
- Busca e filtragem de dados  
- Organização estruturada das informações  

### 🔐 Autenticação e Segurança
- Autenticação com JWT  
- Rotas protegidas  
- Controle de acesso para administradores  

### 📊 Relatórios e Dashboard
- Relatórios gerados no frontend  
- Visualização de dados em tempo real  
- Integração direta com a API  

### 🛠️ Sistema Administrativo
- Cadastro e login de administradores  
- Controle de permissões  
- Operações seguras  

### 📚 Documentação da API
- Integração com Swagger  
- Teste interativo de endpoints  

---

## 🧱 Arquitetura

O projeto segue uma arquitetura organizada e escalável:

### Backend
- Camada de Controllers  
- Camada de Services  
- Repository Pattern  
- Princípios SOLID  

### Frontend
- Arquitetura baseada em componentes  
- Gerenciamento de estado  
- Consumo de API com autenticação  

> 💡 A separação entre frontend e backend garante maior escalabilidade e manutenção.
